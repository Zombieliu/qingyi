# 新成员上手清单

## 1) 环境准备

- [ ] Node 22.x（`nvm use`）
- [ ] 安装依赖：`pnpm install`
- [ ] 复制环境变量：`packages/app/.env.example` → `.env.local`

## 2) 本地运行

- [ ] 启动开发：`pnpm run dev -- --hostname 0.0.0.0 --port 3000`
- [ ] 访问：`http://127.0.0.1:3000`

## 3) 关键路径验证

- [ ] 登录/注册
- [ ] 下单/订单列表
- [ ] 充值页面打开
- [ ] 后台登录与列表打开

## 4) 文档必读

- [ ] `docs/COLLAB_GUIDE.md`
- [ ] `docs/FRONTEND_GUIDE.md`
- [ ] `docs/BACKEND_GUIDE.md`
- [ ] `docs/TEAM_WORKFLOW.md`
