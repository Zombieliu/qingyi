# 发布检查清单

用于每次发布前后逐项确认。

## 发布前

- [ ] `pnpm run lint`
- [ ] `pnpm run build`
- [ ] 关键路径回归（登录/下单/订单/充值）
- [ ] 后台关键页面回归（订单/统计/对账）
- [ ] 若涉及链上：`npm run test:chain:script`
- [ ] DB 迁移文件已提交（如有）
- [ ] 环境变量已检查（见 `ENVIRONMENT_VARIABLES.md`）

## 发布中

- [ ] Vercel 构建成功
- [ ] 部署版本号确认
- [ ] 监控与告警打开

## 发布后

- [ ] 冒烟验证（首页/下单/订单/充值）
- [ ] 后台登录与列表可用
- [ ] 支付回调正常
- [ ] 链上同步 cron 正常
